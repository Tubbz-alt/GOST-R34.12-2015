== Intro
Verilog HDL implementation of the GOST R34.12-2015 - a fresh Russian government standard symmetric key block cipher.

=== Short Chipher Detail

|===============================================
| *Date published* | 2015
| *Structure*      | Substitution-permutation network
| *Key sizes*      | 256 bits
| *Block sizes*    | 128 bits
| *Rounds*         | 10
|===============================================
